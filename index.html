Sections 1-2

순환 없이 모든 토큰 관계를 한 번에 본다

논문의 출발점은 긴 시퀀스에서 RNN의 순차 계산 병목을 줄이는 것이다. 현재 문장 길이 n에 대해 self-attention은 모든 token-to-token 연결을 n x n 격자로 만들고, 가장 먼 두 위치도 한 attention layer 안에서 만난다.

Section 3.1

Encoder-Decoder 스택과 실제 논문 수치

base 모델은 encoder 6층, decoder 6층, d_model 512, feed-forward 2048, attention head 8개를 기준으로 설명된다.

Sections 3.4-3.5

Section 3.2.2

Multi-head attention은 여러 관계 공간을 병렬로 본다

논문 base 모델은 h=8, d_k=d_v=64를 사용한다. 이 데모는 4개 head로 축소했지만, 각 head의 projection이 달라지면 최고 가중치와 분포 엔트로피가 바뀐다.

Section 3.2.3

Decoder는 미래를 가리고 encoder 출력을 다시 본다

autoregressive 생성에서는 현재 위치가 오른쪽의 미래 출력 토큰을 볼 수 없다. 그 다음 encoder-decoder attention에서 target query가 source sentence의 key/value를 참조한다.

Sections 3.3, 3.1

각 위치마다 같은 FFN을 적용하고 residual로 되돌린다

attention이 토큰 사이를 섞는다면, position-wise feed-forward network는 각 토큰 표현 내부를 512 -> 2048 -> 512처럼 확장 후 압축한다.

Section 5

학습 스케줄은 warmup 후 inverse square root로 감소한다

Adam(beta1=0.9, beta2=0.98, eps=1e-9), warmup 4000 step, dropout 0.1, label smoothing 0.1이 base 학습 설정의 핵심이다.

Section 6

결과는 품질과 학습 비용을 같이 본다

논문은 WMT 2014 번역에서 BLEU와 학습 FLOPs를 비교하며, big 모델이 EN-DE 28.4, EN-FR 41.8 BLEU를 보고한다.
